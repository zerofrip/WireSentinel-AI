# WireSentinel-AI

Artificial Intelligence platform for the WireSentinel ecosystem.

## Components

- **Copilot** — Natural-language security assistant
- **Investigation** — AI-driven incident investigation
- **Correlation** — Multi-source threat correlation
- **Knowledge Graph** — Security entity graph
- **RAG** — Retrieval-augmented generation
- **Detections / Policies / Playbooks** — AI-assisted content generation
- **Intelligence** — Threat intel enrichment
- **Analytics / Reporting / Risk** — Summaries and scoring

## Build

```bash
cargo test --workspace
```
